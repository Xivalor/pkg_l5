Назначение программы

Приложение визуализирует работу алгоритмов отсечения отрезков:

Часть 1: отсечение отрезков прямоугольным окном
(алгоритм Сазерленда–Коэна, т.к. вариант 4)

Часть 2: отсечение отрезков выпуклым многоугольником
(так как вариант чётный)

Программа отображает:

систему координат

исходные отрезки

окно / многоугольник

видимые части отрезков после отсечения

Структура проекта
index.html      – интерфейс
style.css       – оформление
main.js         – логика программы и алгоритмы
README.md       – документация

Формат входного файла
n
X1_1 Y1_1 X2_1 Y2_1
...
X1_n Y1_n X2_n Y2_n
Xmin Ymin Xmax Ymax    (для части 1)


Для части 2 (многоугольник):

n
отрезки...
m
X1 Y1
X2 Y2
...
Xm Ym

Как использовать

Открыть index.html в браузере.

Нажать “Загрузить файл” и выбрать входной файл.

На экране появятся:

исходные отрезки (красным)

окно / многоугольник (синим)

Нажать кнопку “Отсечь (Сазерленд–Коэн)” для части 1.

Нажать кнопку “Отсечь многоугольником” для части 2.

Видимые части отрезков будут показаны зелёным.

Краткое описание алгоритмов
Сазерленд–Коэн

Использует 4-битный код положения точки относительно окна.
Позволяет быстро определить:

полностью видимый отрезок

полностью невидимый

частично видимый → вычисляются пересечения со сторонами окна

Отсечение выпуклым многоугольником

Отрезок анализируется относительно каждой грани многоугольника.
Используется параметрическое представление отрезка (t ∈ [0,1]).
После обработки всех граней остаётся видимая часть.

Результат

После выполнения отсечения на холсте отображаются только те части отрезков, которые находятся внутри окна или многоугольника.
